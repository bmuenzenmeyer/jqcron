jqCron - Quartz
======

[![Build Status](https://travis-ci.org/bmuenzenmeyer/jqcron.svg?branch=master)](https://travis-ci.org/bmuenzenmeyer/jqcron)

This is a forked version of [arnapou/jqCron](https://github.com/arnapou/jqcron) and [alexcheng1982/jqCrons](https://github.com/alexcheng1982/jqcron) jQuery plugin using [Quartz](http://quartz-scheduler.org/documentation/quartz-2.x/tutorials/crontrigger) expression.

## Install

Use `bower install jqcron-quartz` to install it.
